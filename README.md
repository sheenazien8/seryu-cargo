# Seryu Cargo Driver Salary

## Description

this is the an example api for salary driver on seryu cargo

## Installation

```bash
$ npm install

# seed the data
$ npm run seed

```

## Setup the env
```bash
$ cp .env.example .env
```
modify the value base on your local configuration

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentation
http://localhost:3000/v1/docs/

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
